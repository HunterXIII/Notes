# 6. –ê–ª–≥–æ—Ä–∏—Ç–º –ï–≤–∫–ª–∏–¥–∞, —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –ï–≤–∫–ª–∏–¥–∞.

**–ê–ª–≥–æ—Ä–∏—Ç–º –ï–≤–∫–ª–∏–¥–∞** - –Ω–∞—Ö–æ–¥–∏—Ç **–Ω–∞–∏–±–æ–ª—å—à–∏–π –æ–±—â–∏–π –¥–µ–ª–∏—Ç–µ–ª—å** –¥–ª—è —á–∏—Å–µ–ª m –∏ n
1. –†–∞–∑–¥–µ–ª–∏–º $m$ –Ω–∞ $n$, –∏ –ø—É—Å—Ç—å –æ—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è –±—É–¥–µ—Ç —Ä–∞–≤–µ–Ω $r$ (–≥–¥–µ $0 ‚â§ r < n$).
2. –ï—Å–ª–∏ $r = 0$, —Ç–æ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –ø—Ä–µ–∫—Ä–∞—â–∞–µ—Ç—Å—è; $n$ ‚Äî –∏—Å–∫–æ–º–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.
3. –ü—Ä–∏—Å–≤–æ–∏—Ç—å $m = n, n = r$ –∏ –≤–µ—Ä–Ω—É—Ç—å—Å—è –∫ —à–∞–≥—É 1.
```python
def euclidus(m, n):
    while n:
        m, n = n, m % n
    return m
```

**–†–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –ï–≤–∫–ª–∏–¥–∞** 
–í —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ "–æ–±—ã—á–Ω—ã–π" –∞–ª–≥–æ—Ä–∏—Ç–º –ï–≤–∫–ª–∏–¥–∞ –ø—Ä–æ—Å—Ç–æ –Ω–∞—Ö–æ–¥–∏—Ç –Ω–∞–∏–±–æ–ª—å—à–∏–π –æ–±—â–∏–π –¥–µ–ª–∏—Ç–µ–ª—å –¥–≤—É—Ö —á–∏—Å–µ–ª m –∏ n, —Ä–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –ï–≤–∫–ª–∏–¥–∞ –Ω–∞—Ö–æ–¥–∏—Ç –ø–æ–º–∏–º–æ –ù–û–î —Ç–∞–∫–∂–µ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã x –∏ y —Ç–∞–∫–∏–µ, —á—Ç–æ:
$$ x * m + y * n = –ù–û–î(m, n) $$
–¢.–µ. –æ–Ω –Ω–∞—Ö–æ–¥–∏—Ç –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã, —Å –ø–æ–º–æ—â—å—é –∫–æ—Ç–æ—Ä—ã—Ö –ù–û–î –¥–≤—É—Ö —á–∏—Å–µ–ª –≤—ã—Ä–∞–∂–∞–µ—Ç—Å—è —á–µ—Ä–µ–∑ —Å–∞–º–∏ —ç—Ç–∏ —á–∏—Å–ª–∞.

```python
def extended_euclidus(m, n):
    if n == 0:
        return m, 1, 0
    d, x1, y1 = extended_euclidus(n, m % n)
    x = y1
    y = x1 - (m // n) * y1
    return d, x, y
```

**–†–µ—à–µ—Ç–æ –≠—Ä–∞—Ç–æ—Å—Ñ–µ–Ω–∞** - –∞–ª–≥–æ—Ä–∏—Ç–º –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏—è –≤—Å–µ—Ö –ø—Ä–æ—Å—Ç—ã—Ö —á–∏—Å–µ–ª –¥–æ –Ω–µ–∫–æ—Ç–æ—Ä–æ–≥–æ —Ü–µ–ª–æ–≥–æ —á–∏—Å–ª–∞ n. 
–ü—Ä–∏–Ω—Ü–∏–ø –ø—Ä–æ—Å—Ç–æ–π: —Å—Ç—Ä–æ–∏–º —Å–ø–∏—Å–æ–∫ —á–∏—Å–µ–ª –æ—Ç 2 –¥–æ n, –ø–æ—Å–ª–µ —á–µ–≥–æ —É–¥–∞–ª—è–µ–º –∏–∑ –Ω–µ–≥–æ –≤—Å–µ —Å–æ—Å—Ç–∞–≤–Ω—ã–µ —á–∏—Å–ª–∞.
–°–æ—Å—Ç–∞–≤–Ω–æ–µ —á–∏—Å–ª–æ - —á–∏—Å–ª–æ, –∏–º–µ—é—â–µ–µ –¥–µ–ª–∏—Ç–µ–ª–∏ –ø–æ–º–∏–º–æ 1 –∏ —á–∏—Å–ª–∞ n. –¢.–µ. –≤—Å–µ —á–∏—Å–ª–∞, –∫—Ä–æ–º–µ –ø—Ä–æ—Å—Ç—ã—Ö.
–ï—Å–ª–∏ –±–æ–ª–µ–µ –ø–æ–¥—Ä–æ–±–Ω–æ, —Ç–æ –∞–ª–≥–æ—Ä–∏—Ç–º –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è —Ä–µ—à–µ—Ç–∞ –≠—Ä–∞—Ç–æ—Å—Ñ–µ–Ω–∞ –≤—ã–≥–ª—è–¥–∏—Ç —Ç–∞–∫:

–ü—É—Å—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è p –∏–∑–Ω–∞—á–∞–ª—å–Ω–æ —Ä–∞–≤–Ω–∞ –¥–≤—É–º ‚Äî –ø–µ—Ä–≤–æ–º—É –ø—Ä–æ—Å—Ç–æ–º—É —á–∏—Å–ª—É.

–ù–∞–π—Ç–∏ –ø–µ—Ä–≤–æ–µ –Ω–µ–∑–∞—á—ë—Ä–∫–Ω—É—Ç–æ–µ —á–∏—Å–ª–æ –≤ —Å–ø–∏—Å–∫–µ, –±–æ–ª—å—à–µ–µ —á–µ–º p, –∏ –ø—Ä–∏—Å–≤–æ–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏—é –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π p —ç—Ç–æ —á–∏—Å–ª–æ.
–ü–æ–≤—Ç–æ—Ä—è—Ç—å —à–∞–≥–∏ 3 –∏ 4, –ø–æ–∫–∞ –≤–æ–∑–º–æ–∂–Ω–æ.

def eratosthenes(n):
    arr = [i for i in range(n + 1)]
    arr[1] = 0
    for p in arr:
        if p != 0:
            arr[2*p::p] = [0] * len(arr[2*p::p])
    return [i for i in arr if i != 0]
```

def insertion_sort(arr: list) -> list:    
    for i in range(1, len(arr)):
        left, right = 0, i - 1
        mid = (left + right) // 2
           
        while left < right:
            if arr[i] == arr[mid]:
                break
            elif arr[i] < arr[mid]:
                right = mid
            else:
                left = mid + 1
            mid = (left + right) // 2
           
        if arr[i] > arr[mid]:
            mid += 1
           
        arr[mid], arr[mid + 1: i + 1] = arr[i], arr[mid:i]
```

 def merge(A, B):
    i, j, C = 0, 0, []
    while True:
        if A[i] < B[j]:
            C.append(A[i])
            i += 1
            if i == len(A):
                C.extend(B[j:])
                break
        else:

                break
    return C

    if len(arr) <= 1:
        return arr
   
    pivot = arr[0]
    left = []
    right = []

        if x < pivot:

def partition(array, low, high):
    pivot = array[high]
    i = low - 1

    for j in range(low, high):
        if array[j] <= pivot:
            i = i + 1
            array[i], array[j] = array[j], array[i]

    array[i + 1], array[high] = array[high], array[i + 1]
    return i + 1


def quicksort(array, low, high):
    if low < high:
        pi = partition(array, low, high)

        quicksort(array, low, pi - 1)
        quicksort(array, pi + 1, high)
```

def levenshtein_distance(s1, s2):
    n, m = len(s1), len(s2)

    if n > m:
        s1, s2 = s2, s1
        n, m = m, n

    current_row = list(range(n + 1))

    for i in range(1, m + 1):
        previous_row, current_row = current_row, [i] + [0] * n
        for j in range(1, n + 1):
            add = previous_row[j] + 1
            delete = current_row[j - 1] + 1
            change = previous_row[j - 1]
            if s1[j - 1] != s2[i - 1]:
                change += 1
            current_row[j] = min(add, delete, change)

    return current_row[n]
```

**–ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã** (Dijkstra‚Äôs algorithm) ‚Äî –∞–ª–≥–æ—Ä–∏—Ç–º –Ω–∞ –≥—Ä–∞—Ñ–∞—Ö, –∏–∑–æ–±—Ä–µ—Ç—ë–Ω–Ω—ã–π –Ω–∏–¥–µ—Ä–ª–∞–Ω–¥—Å–∫–∏–º —É—á—ë–Ω—ã–º –≠–¥—Å–≥–µ—Ä–æ–º –î–µ–π–∫—Å—Ç—Ä–æ–π –≤ 1959 –≥–æ–¥—É. –ù–∞—Ö–æ–¥–∏—Ç –∫—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –æ—Ç –æ–¥–Ω–æ–π –∏–∑ –≤–µ—Ä—à–∏–Ω –≥—Ä–∞—Ñ–∞ –¥–æ –≤—Å–µ—Ö –æ—Å—Ç–∞–ª—å–Ω—ã—Ö. –ê–ª–≥–æ—Ä–∏—Ç–º —Ä–∞–±–æ—Ç–∞–µ—Ç —Ç–æ–ª—å–∫–æ –¥–ª—è –≥—Ä–∞—Ñ–æ–≤ –±–µ–∑ —Ä—ë–±–µ—Ä –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –≤–µ—Å–∞. –ê–ª–≥–æ—Ä–∏—Ç–º —à–∏—Ä–æ–∫–æ –ø—Ä–∏–º–µ–Ω—è–µ—Ç—Å—è –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏, –Ω–∞–ø—Ä–∏–º–µ—Ä, –µ–≥–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –ø—Ä–æ—Ç–æ–∫–æ–ª—ã –º–∞—Ä—à—Ä—É—Ç–∏–∑–∞—Ü–∏–∏ OSPF –∏ IS-IS.

## –ö—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –≤–æ –≤–∑–≤–µ—à–µ–Ω–Ω—ã—Ö –≥—Ä–∞—Ñ–∞—Ö. –ê–ª–≥–æ—Ä–∏—Ç–º –î–µ–π–∫—Å—Ç—Ä—ã

1. –°–æ–∑–¥–∞—Ç—å –º–∞—Å—Å–∏–≤ $dist$ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–π. –ò–∑–Ω–∞—á–∞–ª—å–Ω–æ $dist[s]=0$ –∏ $dist[v]=‚àû$ –¥–ª—è $v‚â†s$.
2. –°–æ–∑–¥–∞—Ç—å –±—É–ª–µ–≤ –º–∞—Å—Å–∏–≤ $used$, $used[v]=0$ –¥–ª—è –≤—Å–µ—Ö –≤–µ—Ä—à–∏–Ω $v$ ‚Äî –≤ –Ω—ë–º –º—ã –±—É–¥–µ–º –æ—Ç–º–µ—á–∞—Ç—å, —Å–æ–≤–µ—Ä—à–∞–ª–∞—Å—å –ª–∏ —Ä–µ–ª–∞–∫—Å–∞—Ü–∏—è –∏–∑ –≤–µ—Ä—à–∏–Ω—ã.
3. –ü–æ–∫–∞ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –≤–µ—Ä—à–∏–Ω–∞ $v$ —Ç–∞–∫–∞—è, —á—Ç–æ $used[v]=0$ –∏ $dist[v]‚â†‚àû$, –ø—Ä–∏—Ç–æ–º, –µ—Å–ª–∏ —Ç–∞–∫–∏—Ö –≤–µ—Ä—à–∏–Ω –Ω–µ—Å–∫–æ–ª—å–∫–æ, —Ç–æ $v$ ‚Äî –≤–µ—Ä—à–∏–Ω–∞ —Å –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–º $dist[v]$, –¥–µ–ª–∞—Ç—å —Å–ª–µ–¥—É—é—â–µ–µ:
    1. –ü–æ–º–µ—Ç–∏—Ç—å, —á—Ç–æ –º—ã —Å–æ–≤–µ—Ä—à–∞–ª–∏ —Ä–µ–ª–∞–∫—Å–∞—Ü–∏—é –∏–∑ –≤–µ—Ä—à–∏–Ω—ã $v$, —Ç–æ –µ—Å—Ç—å –ø—Ä–∏—Å–≤–æ–∏—Ç—å $used[v]=1$.
    2. –†–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞–µ–º –≤—Å–µ —Ä—ë–±—Ä–∞ $(v,u)‚ààE$. –î–ª—è –∫–∞–∂–¥–æ–≥–æ —Ä–µ–±—Ä–∞ –ø—ã—Ç–∞–µ–º—Å—è —Å–¥–µ–ª–∞—Ç—å —Ä–µ–ª–∞–∫—Å–∞—Ü–∏—é: –µ—Å–ª–∏ $dist[v]+w(v,u)<dist[u]$, –ø—Ä–∏—Å–≤–æ–∏—Ç—å $dist[u]=dist[v]+w(v,u)$.

–í—ã—á–∏—Å–ª–∏–º –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –∞–ª–≥–æ—Ä–∏—Ç–º–∞. –ú—ã $V$ —Ä–∞–∑ –∏—â–µ–º –≤–µ—Ä—à–∏–Ω—É –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–º $dist$, –ø–æ–∏—Å–∫ –º–∏–Ω–∏–º—É–º–∞ –ª–∏–Ω–µ–π–Ω—ã–π –∑–∞ $O(V)$, –æ—Ç—Å—é–¥–∞ $O(V^2)$. –û–±—Ä–∞–±–æ—Ç–∫–∞ —Ä–µ–±–µ—Ä –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Å—É–º–º–∞—Ä–Ω–æ –∑–∞ $O(E)$, –ø–æ—Ç–æ–º—É —á—Ç–æ –Ω–∞ –∫–∞–∂–¥–æ–µ —Ä–µ–±—Ä–æ –º—ã —Ç—Ä–∞—Ç–∏–º $O(1)$ –¥–µ–π—Å—Ç–≤–∏–π. 
**–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —Ñ–∏–Ω–∞–ª—å–Ω–∞—è –∞—Å–∏–º–ø—Ç–æ—Ç–∏–∫–∞: $O(V^2+E)$.**

–ò—Å–∫–∞—Ç—å –≤–µ—Ä—à–∏–Ω—É —Å –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–º $dist$ –º–æ–∂–Ω–æ –≥–æ—Ä–∞–∑–¥–æ –±—ã—Å—Ç—Ä–µ–µ, –∏—Å–ø–æ–ª—å–∑—É—è —Ç–∞–∫—É—é —Å—Ç—Ä—É–∫—Ç—É—Ä—É –¥–∞–Ω–Ω—ã—Ö –∫–∞–∫ –æ—á–µ—Ä–µ–¥—å —Å –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º. 

–ò—Å–∫–∞—Ç—å –≤–µ—Ä—à–∏–Ω—É —Å –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–º $dist$ –º–æ–∂–Ω–æ –≥–æ—Ä–∞–∑–¥–æ –±—ã—Å—Ç—Ä–µ–µ, –∏—Å–ø–æ–ª—å–∑—É—è —Ç–∞–∫—É—é —Å—Ç—Ä—É–∫—Ç—É—Ä—É –¥–∞–Ω–Ω—ã—Ö –∫–∞–∫ –æ—á–µ—Ä–µ–¥—å —Å –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–º. –ù–∞–º –Ω—É–∂–Ω–æ —Ö—Ä–∞–Ω–∏—Ç—å –ø–∞—Ä—ã $(dist,index)$ –∏ —É–º–µ—Ç—å –¥–µ–ª–∞—Ç—å —Ç–∞–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏:

- –ò–∑–≤–ª–µ—á—å –º–∏–Ω–∏–º—É–º (—á—Ç–æ–±—ã –æ–±—Ä–∞–±–æ—Ç–∞—Ç—å –Ω–æ–≤—É—é –≤–µ—Ä—à–∏–Ω—É)
- –£–¥–∞–ª–∏—Ç—å –≤–µ—Ä—à–∏–Ω—É –ø–æ –∏–Ω–¥–µ–∫—Å—É (—á—Ç–æ–±—ã —É–º–µ–Ω—å—à–∏—Ç—å $dist$ –¥–æ –∫–∞–∫–æ–≥–æ-—Ç–æ —Å–æ—Å–µ–¥–∞)
- –î–æ–±–∞–≤–∏—Ç—å –Ω–æ–≤—É—é –≤–µ—Ä—à–∏–Ω—É (—á—Ç–æ–±—ã —É–º–µ–Ω—å—à–∏—Ç—å $dist$ –¥–æ –∫–∞–∫–æ–≥–æ-—Ç–æ —Å–æ—Å–µ–¥–∞)

–î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É—é—Ç, –Ω–∞–ø—Ä–∏–º–µ—Ä, –∫—É—á—É –∏–ª–∏ —Å–µ—Ç. –£–¥–æ–±–Ω–æ –ø–æ–º–∏–º–æ —Å–µ—Ç–∞ —Ö—Ä–∞–Ω–∏—Ç—å —Å–∞–º –º–∞—Å—Å–∏–≤ dist, –∫–æ—Ç–æ—Ä—ã–π –µ–≥–æ –¥—É–±–ª–∏—Ä—É–µ—Ç, –Ω–æ —Ö—Ä–∞–Ω–∏—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –ø–æ –ø–æ—Ä—è–¥–∫—É. –¢–æ–≥–¥–∞, —á—Ç–æ–±—ã –∑–∞–º–µ–Ω–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ $(dist1,u)$ –Ω–∞ $(dist2,u)$, –Ω—É–∂–Ω–æ —É–¥–∞–ª–∏—Ç—å –∏–∑ —Å–µ—Ç–∞ –∑–Ω–∞—á–µ–Ω–∏–µ $(dist[u],u)$, —Å–¥–µ–ª–∞—Ç—å $dist[u]=dist2$; –∏ –¥–æ–±–∞–≤–∏—Ç—å –≤ —Å–µ—Ç $(dist[u],u)$.

–î–∞–Ω–Ω—ã–π –∞–ª–≥–æ—Ä–∏—Ç–º –±—É–¥–µ—Ç —Ä–∞–±–æ—Ç–∞—Ç—å –∑–∞ $O(\log{V})$ –∏–∑–≤–ª–µ—á–µ–Ω–∏–π –º–∏–Ω–∏–º—É–º–∞ –∏ $O(E\log{V})$ –æ–ø–µ—Ä–∞—Ü–∏–π —É–º–µ–Ω—å—à–µ–Ω–∏—è —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è –¥–æ –≤–µ—Ä—à–∏–Ω—ã. –ü–æ—ç—Ç–æ–º—É –∞–ª–≥–æ—Ä–∏—Ç–º —Ä–∞–±–æ—Ç–∞–µ—Ç –∑–∞ $O(E\log{V})$.

–ó–∞–º–µ—Ç—å—Ç–µ, —á—Ç–æ —ç—Ç–æ—Ç –∞–ª–≥–æ—Ä–∏—Ç–º –Ω–µ –ª—É—á—à–µ –∏ –Ω–µ —Ö—É–∂–µ –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –±–µ–∑ —Å–µ—Ç–∞, –∫–æ—Ç–æ—Ä—ã–π —Ä–∞–±–æ—Ç–∞–µ—Ç –∑–∞ $O(V^2+E)$. –í–µ–¥—å –µ—Å–ª–∏ $E=O(V^2)$ (–≥—Ä–∞—Ñ –ø–æ—á—Ç–∏ –ø–æ–ª–Ω—ã–π), —Ç–æ –î–µ–π–∫—Å—Ç—Ä–∞ –±–µ–∑ —Å–µ—Ç–∞ —Ä–∞–±–æ—Ç–∞–µ—Ç –±—ã—Å—Ç—Ä–µ–µ, –∞ –µ—Å–ª–∏, –Ω–∞–ø—Ä–∏–º–µ—Ä, $E=O(V)$, —Ç–æ –î–µ–π–∫—Å—Ç—Ä–∞ –Ω–∞ —Å–µ—Ç–µ —Ä–∞–±–æ—Ç–∞–µ—Ç –±—ã—Å—Ç—Ä–µ–µ.

# 28. –ö—Ä–∞—Ç—á–∞–π—à–∏–µ –ø—É—Ç–∏ –≤ –∞—Ü–∏–∫–ª–∏—á–µ—Å–∫–∏—Ö –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –≥—Ä–∞—Ñ–∞—Ö. 

–ü—É—Å—Ç—å –¥–∞–Ω –∞—Ü–∏–∫–ª–∏—á–µ—Å–∫–∏–π –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –≤–∑–≤–µ—à–µ–Ω–Ω—ã–π –≥—Ä–∞—Ñ. –¢—Ä–µ–±—É–µ—Ç—Å—è –Ω–∞–π—Ç–∏ –≤–µ—Å –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –∏–∑ $u$ –≤ $v$.

–ü—É—Å—Ç—å $d$ ‚Äî —Ñ—É–Ω–∫—Ü–∏—è, –≥–¥–µ $d(i)$ ‚Äî –≤–µ—Å –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –∏–∑ $u$ –≤ $i$. –Ø—Å–Ω–æ, —á—Ç–æ $d(u)$ —Ä–∞–≤–µ–Ω $0$. –ü—É—Å—Ç—å $w(i,j)$ ‚Äî –≤–µ—Å —Ä–µ–±—Ä–∞ –∏–∑ $i$ –≤ $j$. –ë—É–¥–µ–º –æ–±—Ö–æ–¥–∏—Ç—å –≥—Ä–∞—Ñ –≤ –ø–æ—Ä—è–¥–∫–µ —Ç–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏. –ü–æ–ª—É—á–∞–µ–º —Å–ª–µ–¥—É—é—â–∏–µ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏—è: $$ d(i)=\min{(d(i),d(j)+w(j,i))} $$ –¢–∞–∫ –∫–∞–∫ –º—ã –æ–±—Ö–æ–¥–∏–º –≥—Ä–∞—Ñ –≤ –ø–æ—Ä—è–¥–∫–µ —Ç–æ–ø–æ–ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å–æ—Ä—Ç–∏—Ä–æ–≤–∫–∏, —Ç–æ –Ω–∞ $i$-–æ–º —à–∞–≥–µ –≤—Å–µ–º $d(j)$ ($j$ —Ç–∞–∫–∏–µ, —á—Ç–æ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç —Ä–µ–±—Ä–æ –∏–∑ $j$ –≤ $i$) —É–∂–µ –ø—Ä–∏—Å–≤–æ–µ–Ω—ã –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–µ –æ—Ç–≤–µ—Ç—ã, –∏, —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, $d(i)$ —Ç–∞–∫–∂–µ –±—É–¥–µ—Ç –ø—Ä–∏—Å–≤–æ–µ–Ω –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã–π –æ—Ç–≤–µ—Ç.

# 29. –ê–ª–≥–æ—Ä–∏—Ç–º –ë–µ–ª–ª–º–∞–Ω–∞-–§–æ—Ä–¥–∞, –ø—Ä–æ–≤–µ—Ä–∫–∞ –Ω–∞–ª–∏—á–∏—è —Ü–∏–∫–ª–∞ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–≥–æ –≤–µ—Å–∞. 

**–ê–ª–≥–æ—Ä–∏—Ç–º –ë–µ–ª–ª–º–∞–Ω–∞-–§–æ—Ä–¥–∞**¬†–ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω –¥–ª—è —Ä–µ—à–µ–Ω–∏—è –∑–∞–¥–∞—á–∏ –ø–æ–∏—Å–∫–∞ –∫—Ä–∞—Ç—á–∞–π—à–µ–≥–æ –ø—É—Ç–∏ –Ω–∞ –≥—Ä–∞—Ñ–µ. –î–ª—è –∑–∞–¥–∞–Ω–Ω–æ–≥–æ –æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –≤–∑–≤–µ—à–µ–Ω–Ω–æ–≥–æ –≥—Ä–∞—Ñ–∞ –∞–ª–≥–æ—Ä–∏—Ç–º –Ω–∞—Ö–æ–¥–∏—Ç –∫—Ä–∞—Ç—á–∞–π—à–∏–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏—è –æ—Ç –≤—ã–¥–µ–ª–µ–Ω–Ω–æ–π –≤–µ—Ä—à–∏–Ω—ã-–∏—Å—Ç–æ—á–Ω–∏–∫–∞ –¥–æ –≤—Å–µ—Ö –æ—Å—Ç–∞–ª—å–Ω—ã—Ö –≤–µ—Ä—à–∏–Ω –≥—Ä–∞—Ñ–∞.

–ê–ª–≥–æ—Ä–∏—Ç–º –ë–µ–ª–ª–º–∞–Ω–∞-–§–æ—Ä–¥–∞ –º–∞—Å—à—Ç–∞–±–∏—Ä—É–µ—Ç—Å—è —Ö—É–∂–µ –¥—Ä—É–≥–∏—Ö –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤ —Ä–µ—à–µ–Ω–∏—è —É–∫–∞–∑–∞–Ω–Ω–æ–π –∑–∞–¥–∞—á–∏ (—Å–ª–æ–∂–Ω–æ—Å—Ç—å $O(|V||E|)$ –ø—Ä–æ—Ç–∏–≤ $O(|E|+|V|\ln{|V|})$ —É –∞–ª–≥–æ—Ä–∏—Ç–º–∞ –î–µ–π–∫—Å—Ç—Ä—ã), –æ–¥–Ω–∞–∫–æ –µ–≥–æ –æ—Ç–ª–∏—á–∏—Ç–µ–ª—å–Ω–æ–π –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç—å—é —è–≤–ª—è–µ—Ç—Å—è –ø—Ä–∏–º–µ–Ω–∏–º–æ—Å—Ç—å –∫ –≥—Ä–∞—Ñ–∞–º —Å –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–º–∏, –≤ —Ç–æ–º —á–∏—Å–ª–µ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º–∏, –≤–µ—Å–∞–º–∏.

1. –ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—è: –≤—Å–µ–º –≤–µ—Ä—à–∏–Ω–∞–º –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ–º–æ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ $dist[v]=‚àû$, –∫—Ä–æ–º–µ –≤–µ—Ä—à–∏–Ω—ã-–∏—Å—Ç–æ—á–Ω–∏–∫–∞, –¥–ª—è –∫–æ—Ç–æ—Ä–æ–π $dist(u)=0$.
2. –†–µ–ª–∞–∫—Å–∞—Ü–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤–∞ —Ä—ë–±–µ—Ä $E$
    1. –î–ª—è –∫–∞–∂–¥–æ–≥–æ —Ä–µ–±—Ä–∞ $e=(v,z)‚ààE$ –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –Ω–æ–≤–æ–µ –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ–º–æ–µ —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ $new_dist(z)=dist(v)+w(e)$.
    2. –ï—Å–ª–∏ $new_dist(z)<dist(z)$, —Ç–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ $dist(z)=new_dist(z)$ (—Ä–µ–ª–∞–∫—Å–∞—Ü–∏—è —Ä–µ–±—Ä–∞ $e$).
